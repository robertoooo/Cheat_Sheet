# Django Basics
```sh
django-admin.py startproject project_name #creates a new project
python manage.py startapp name #Creates a django app directory structure 

python manage.py test #Discover and run tests in the specified modules or the current directory
python manage.py runserver #run the server
python manage.py migrate #update database schema

python manage.py makemgrations core #Migrate the database schema on the app core
```
